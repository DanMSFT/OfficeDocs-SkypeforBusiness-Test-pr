# Installing Persistent Chat Server in Lync Server 2013

 


Installing Persistent Chat Server is integrated into Lync Server 2013 setup and uses the same Bootstrapper that Lync Server 2013 uses for installation. For installation instructions, see [Deploying Lync Server 2013](lync-server-2013-deploying-lync-server.md) in the Deployment documentation.
